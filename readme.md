# 🚀 Ultimate Algo Trading Bot (FastAPI + MT5)

ระบบเทรดอัตโนมัติ (Algorithmic Trading) ขั้นสูงที่เชื่อมต่อกับ MetaTrader 5 (MT5) ขับเคลื่อนด้วย Python และควบคุมผ่าน Web Dashboard สไตล์ Modern สุดล้ำ (Microservices Architecture) พร้อมระบบรักษาความปลอดภัยและการคำนวณสภาวะตลาดอัจฉริยะ

---

## ✨ ฟีเจอร์เด่น (Key Features)

* 🔒 **Security Login (SPA Auth):** ระบบล็อคหน้าเว็บด้วยรหัสผ่าน ป้องกันผู้ไม่หวังดีเข้าถึงพอร์ตของคุณ (รหัสผ่านเก็บปลอดภัยในไฟล์ `.env`)
* 💰 **Real-time PnL & Account Summary:** แสดงยอด Balance, Equity, และกำไร/ขาดทุนรวมของพอร์ต รวมถึงกำไร/ขาดทุนของแต่ละออร์เดอร์แบบ Real-time บนหน้าเว็บ
* 🛑 **Manual Close (Panic Button):** ปุ่ม "ปิดออร์เดอร์" ฉุกเฉินบนหน้าเว็บ กดปุ๊บบอทสั่งปิดทำกำไร/ตัดขาดทุนให้ทันทีโดยไม่ต้องเปิด MT5
* 📊 **Live Indicator Dashboard:** แสดงราคาปัจจุบัน, สัญญาณ (BUY/SELL/WAIT), และค่าอินดิเคเตอร์ (ADX, RSI) ให้ดูแบบสดๆ ในหน้าการ์ดของแต่ละคู่เงิน
* 🧠 **Auto-Detect Market Regime:** โหมดอัจฉริยะที่ให้บอทวิเคราะห์ความแรงของเทรนด์ (ADX) และสลับกลยุทธ์เองอัตโนมัติ (เช่น มีเทรนด์ใช้เส้นตัดกัน, ไซด์เวย์ใช้ดักสวิงกรอบ)
* ⚙️ **5 Built-in Strategies:** เลือกเปลี่ยนกลยุทธ์ให้แต่ละคู่เงินได้อิสระตลอดเวลาโดยไม่ต้องรีสตาร์ทบอท
* 🛡️ **Auto-Calculate Risk:** ระบบคำนวณ Lot Size และระยะ TP/SL ให้อัตโนมัติตามความเสี่ยง (% Risk) และค่า Spread ของโบรกเกอร์เมื่อเพิ่มคู่เงินใหม่
* 📱 **Telegram Notifications:** แจ้งเตือนทุกความเคลื่อนไหว (เข้าออร์เดอร์, ปิดออร์เดอร์ด้วยมือ, ขยับ Trailing Stop) เข้ามือถือทันที
* 💾 **SQLite Database:** บันทึกการตั้งค่าพอร์ตโฟลิโออัตโนมัติ ปิดคอมเปิดใหม่ค่าไม่หาย

## 🛠️ สิ่งที่ต้องเตรียมก่อนติดตั้ง (Prerequisites)
1. **OS:** ระบบปฏิบัติการ **Windows** เท่านั้น (เนื่องจากไลบรารี MetaTrader5 รองรับเฉพาะ Windows)
2. **Python:** ติดตั้ง Python เวอร์ชั่น 3.9 ขึ้นไป (ตอนติดตั้งอย่าลืมติ๊กถูกที่ `Add Python to PATH`)
3. **MetaTrader 5 (MT5):** ติดตั้งโปรแกรม MT5 และล็อกอินเข้าสู่บัญชีเทรด (Demo หรือ Real) ให้เรียบร้อย
4. **Telegram Bot:** เตรียม `Bot Token` และ `Chat ID` สำหรับการแจ้งเตือน

## ⚙️ ขั้นตอนการติดตั้ง (Installation)

**1. สร้างโฟลเดอร์โปรเจ็กต์และเปิด Terminal (Command Prompt / VS Code)**

**2. สร้างและเปิดใช้งาน Virtual Environment**
เพื่อป้องกันไลบรารีชนกัน ให้รันคำสั่ง:
```cmd
python -m venv env
env\Scripts\activate

```

**3. ติดตั้งไลบรารีที่จำเป็น (Dependencies)**
คัดลอกคำสั่งนี้ไปรันเพื่อติดตั้งเครื่องมือทั้งหมด:

```cmd
pip install MetaTrader5 pandas pandas_ta fastapi uvicorn jinja2 python-multipart requests python-dotenv

```

## 🚀 การตั้งค่าและการใช้งาน (How to Run)

**1. สร้างไฟล์รหัสผ่าน (`.env`)**
สร้างไฟล์ชื่อ `.env` ในโฟลเดอร์โปรเจ็กต์ และใส่ข้อมูลของคุณลงไปดังนี้:

```env
TELEGRAM_TOKEN=ใส่_TOKEN_ของคุณตรงนี้
TELEGRAM_CHAT_ID=ใส่_CHAT_ID_ของคุณตรงนี้
BOT_PASSWORD=รหัสผ่านเข้าเว็บของคุณ (เช่น admin1234)

```

**2. ตั้งค่า MT5**
เปิดโปรแกรม MT5 ไปที่เมนู `Tools` -> `Options` -> แท็บ `Expert Advisors` -> ติ๊กถูกที่ **"Allow algorithmic trading"**

**3. สั่งรันเซิร์ฟเวอร์บอท**
**ดับเบิ้ลคลิกไฟล์ `run_server.bat**` (หน้าต่าง Terminal สีดำ-เขียว จะเด้งขึ้นมาเพื่อรันระบบ Backend)

**4. เข้าสู่แผงควบคุม (Web Dashboard)**

* เปิดเว็บบราวเซอร์ (Chrome, Edge)
* พิมพ์ URL: `http://127.0.0.1:8000`
* **กรอกรหัสผ่าน** ที่คุณตั้งไว้ในไฟล์ `.env` เพื่อปลดล็อคหน้าจอ
* คุณสามารถเพิ่มคู่เงิน (บอทจะคำนวณค่าให้และตั้งเป็นโหมด `AUTO_DETECT` อัตโนมัติ), ปรับเปลี่ยนตั้งค่า, กดปิดออร์เดอร์ฉุกเฉิน, และกดปุ่ม **"▶️ เริ่มทำงานบอท"** จากหน้าเว็บได้เลย!

## 🧠 คำอธิบายกลยุทธ์ (Strategies Guide)

* `AUTO_DETECT`: **(แนะนำ)** บอทจะเช็คค่าความแรงเทรนด์ (ADX) ถ้าระดับ > 25 จะเลือกใช้ `Trend_ADX_EMA` ถ้าต่ำกว่า 25 จะสลับไปดักสวิงด้วย `Bollinger_Bands` ให้เองแบบอัตโนมัติ
* `Trend_ADX_EMA`: สายตามเทรนด์ (เหมาะกับกราฟวิ่งเป็นเทรนด์ชัดเจน)
* `MACD_EMA200`: สายกินคำโต (เหมาะกับทองคำและคริปโตที่วิ่งยาวๆ)
* `Bollinger_Bands`: สายสวิงเทรด (เหมาะกับตลาดไซด์เวย์ วิ่งออกข้าง)
* `Donchian_Breakout`: สายระเบิดทะลุกรอบ (เหมาะกับช่วงข่าวแรงๆ หรือเบรกเอาท์)
* `Stoch_RSI`: สายฉาบฉวยสแคลปปิ้ง (เหมาะกับการหาจุดกลับตัวสั้นๆ)

### 🧠 ความเข้าใจพื้นฐาน: "ระยะจุด (Points)" ของคริปโต

ใน MT5 ระยะทางที่กราฟวิ่งจะนับเป็น **"จุด (Points)"** ครับ
สำหรับ Forex (เช่น EURUSD) วิ่ง 100-500 จุดถือว่าเยอะแล้ว **แต่สำหรับคริปโต (BTC, ETH) กราฟขยับแค่วินาทีเดียวก็เป็น 1,000 จุดแล้วครับ!** ดังนั้นตัวเลข TP/SL ของคริปโตจะต้องตั้งเป็น "หลักพัน หรือ หลักหมื่น" เสมอครับ

---

### ⚙️ อธิบาย 5 ช่องตั้งค่าแบบฉบับสายซิ่ง (Scalping)

**1. Strategy Setup (กลยุทธ์)**

* **แนะนำให้เลือก:** `Scalping_Fast`
* **เหตุผล:** เราเพิ่งเขียนสูตรนี้เข้าไปเพื่อเล่นกราฟ 5 นาที (M5) โดยเฉพาะ มันจะใช้เส้น EMA 5 ตัด 13 ซึ่งไวต่อราคามากๆ เข้าไวออกไว ไม่แช่ออร์เดอร์นานครับ

**2. Risk (%) (ความเสี่ยงต่อ 1 ไม้)**

* **คืออะไร:** บอทจะคำนวณ Lot Size ให้อัตโนมัติ โดยอิงจากเงินในกระเป๋า (Balance) และระยะ Stop Loss ครับ
* **แนะนำให้ตั้ง:** **`1`** หรือ **`2`** (หมายถึง ถ้าบอทเทรดแพ้ไม้หลุด SL คุณจะเสียเงินแค่ 1% ของพอร์ตเท่านั้น พอร์ตไม่แตกแน่นอน!)

**3. Stop Loss (SL - จุดตัดขาดทุน)**

* **คืออะไร:** ระยะยอมแพ้ ถ้ากราฟวิ่งผิดทาง บอทจะตัดขาดทุนทันทีเพื่อรักษาชีวิตพอร์ต
* **หลักการตั้งสายซิ่ง:** ต้องตั้งให้ "กว้างกว่า TP เล็กน้อย" เพื่อป้องกันกราฟสะบัดกิน SL ก่อนที่จะวิ่งไปหาเป้าหมายครับ

**4. Take Profit (TP - จุดเก็บกำไร)**

* **คืออะไร:** ระยะทำกำไร กราฟวิ่งชนปุ๊บ ปิดออร์เดอร์รับเงินเข้ากระเป๋าทันที
* **หลักการตั้งสายซิ่ง:** ตั้งให้ "สั้นๆ" เน้นชนไวๆ เก็บกำไรรอบละนิดแต่ได้หลายๆ รอบครับ

**5. Trailing (ระบบเลื่อนกันหน้าทุน)** 🌟 *(ทีเด็ดของบอทเรา)*

* **คืออะไร:** ถ้าระบุตัวเลขลงไป เมื่อกราฟวิ่งกำไรไปถึงระยะที่คุณตั้งไว้ บอทจะ **"เลื่อน SL มาบังหน้าทุน"** (หรือล็อคกำไร) ให้แบบอัตโนมัติ!
* **ข้อดี:** สมมติคุณเข้า BUY แล้วกราฟพุ่งขึ้นไป บอทจะขยับเส้น SL ตามก้นขึ้นไปเรื่อยๆ ถ้ากราฟทุบลงมาอย่างกะทันหัน คุณก็ยังได้กำไรหรืออย่างแย่สุดคือเท่าทุน (เสมอตัว) ครับ!
